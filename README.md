# Can Asistan 🎙️📱

Can Asistan, Android cihazlar için geliştirilmiş, sesli komutlarla mesaj gönderen, arama yapan ve cihaz bilgilerini paylaşan akıllı bir yardımcı uygulamadır. 

Özellikle erişilebilirlik odaklı tasarımı ve arka planda çalışma yeteneği ile eller serbest bir deneyim sunar.

## ✨ Özellikler

- **📞 Sesli Arama:** "Annemi ara" gibi komutlarla rehberden otomatik arama başlatma.
- **📩 Akıllı SMS:** 
  - Sesli mesaj yazma ve gönderme.
  - Gelen mesajları sesli okuma.
  - Mesajları "okundu" olarak işaretleme.
- **🔔 Gelen Çağrı/SMS Duyurusu:** Telefon çaldığında veya mesaj geldiğinde arayanın ismini sesli duyurma.
- **🔋 Cihaz Bilgisi:** Pil seviyesi, saat ve tarih sorgulama.
- **🔢 Sesli Hesap Makinesi:** "15 artı 25 kaç eder?" gibi basit matematiksel işlemleri yapabilme.
- **🔇 Akıllı Ses Yönetimi (Audio Focus):** Asistan konuşurken zil sesini veya müziği otomatik kısma.

## 🛠️ Teknik Detaylar

- **Dil:** Kotlin
- **Minimum SDK:** API 24 (Android 7.0)
- **Hedef SDK:** API 34 (Android 14)
- **Kütüphaneler:** 
  - `SpeechRecognizer` (Ses Tanıma)
  - `TextToSpeech` (Sesli Yanıt)
  - `NotificationListenerService` (Bildirim Yönetimi)
